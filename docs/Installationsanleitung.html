Installationsanleitung für Gambio GX3

Hinweis: Die folgende Installationsanleitung richtet sich an Shopbetreiber, die das releva.nz Plugin selbst installieren möchten.

Modul installieren

  1. Kopieren Sie mit Ihrem FTP-Programm das Verzeichnis Releva in das Verzeichnis GXModules von Ihrem Shop.
  2. Wenn Sie im Shop-Backend eine entsprechende Meldung sehen, passen Sie mit Ihrem FTP-Programm die Datei-Berechtigungen von dem Relevanz Verzeichnis und allen Unterverzeichnissen und Dateien so an, dass der Webserver die Dateien lesen und schreiben kann.

Modul aktivieren

  1. Loggen Sie sich in den Shop Admin Ihres Webshops ein.
  2. Leeren Sie den Cache
    1. Öffnen Sie im Menü unter "Toolbox" die Seite "Cache".
    2. Klicken Sie auf die Schaltfläche "Ausführen" von den Zeilen "Cache für Modulinformationen leeren" und "Cache für Texte leeren" (Reihenfolge beachten!).
  3. Öffnen Sie im Menü unter "Module" das "Modul-Center".
  4. Wählen Sie In der Liste das Modul "releva.nz" an.
  5. Klicken Sie Auf die Schaltfläche "Installieren".
  6. Das Modul befindet sich nun in der Navigation unter "releva.nz".

Modul einrichten

  1. Klicken Sie in der Navigation auf "releva.nz" und anschließend auf "Konfiguration".
  2. Hinterlegen Sie in dem entsprechenden Eingabefeld den von uns bereitgestellten API-Key.
  3. Klicken Sie auf die Schaltfläche "Speichern".
  4. Wenn der API-Key korrekt ist, wird Ihnen eine entsprechende Erfolgsmeldung angezeigt. Falls nicht, überprüfen Sie bitte, ob Sie den API-Key korrekt eingegeben haben.
  5. Die Einrichtung ist nun erfolgreich abgeschloßen. Sie haben nun Zugriff auf Statistiken und die Konfiguration ihres Werbebudgets unter dem Menüpunkt "releva.nz" → "Statistik".
